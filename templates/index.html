Slack Poll App

A simple Slack application for creating and managing polls.

Features:

  • Create polls with multiple options using the /poll slash command
  • Set optional deadlines for automatic poll closure
  • Configure multiple votes per user
  • Hide votes until the poll is closed
  • Manually close polls and publish results

How to Use:

  1. Add this app to your Slack workspace
  2. Use the /poll command in any channel
  3. Fill out the poll creation form
  4. Share the poll with your team!
This application is currently running and ready to handle Slack events.
